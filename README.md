# nft_marketplace
지역주도형 NFT marketplace

## 📋 Introduce
서버로부터 손톱 및 손 이미지 데이터를 요청하여 NFT로 만들고, 판매, 구매, 구매한 목록을 조회할 수 있는 마켓플레이스 서비스입니다.
